## 🚨 **What is Ollama Folder Analyzer ?**

Ollama Folder Analyzer is a command-line tool that scans local project folders and analyzes them using a local Large Language Model (LLM) running through Ollama. It automatically extracts a concise JSON report for each folder (topic, summary, keywords, and detected languages) based on a short textual snapshot of its files and subfolders. All analyses are stored in a lightweight SQLite database, so you can query topics, export results to CSV, and build your own reports on top of the data.

By combining local LLMs with simple folder scanning, Ollama Folder Analyzer gives you a fast, privacy‑friendly way to explore large codebases, discover relevant projects, and keep an indexed overview of what lives on your disk.
